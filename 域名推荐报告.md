## 四、备选 TLD 推荐

如果上述 `.com` 均被注册，优先考虑以下 TLD：

| TLD | 适用场景 | 推荐域名示例 | 优劣势 |
|-----|---------|------------|--------|
| **.io** | SaaS/工具类 | floorplan.io, roomlayout.io | ✅ 科技感强，开发者熟悉；❌ 非传统，部分用户不信任 |
| **.co** | 创业公司 | floorplan.co, aiplan.co | ✅ 简洁，国际化；❌ 易与.com混淆 |
| **.app** | 应用/工具 | floorplan.app, roomlayout.app | ✅ Google主推，HTTPS强制；❌ 较新，认知度待提升 |
| **.ai** | AI 产品 | floorplan.ai, roomdesign.ai | ✅ 精准定位AI产品；❌ 价格较高（~$100+/年） |
| **.design** | 设计类 | floorplan.design, room.design | ✅ 垂直定位准确；❌ 受众较窄 |
| **.studio** | 创意工具 | floorplan.studio, room.studio | ✅ 创意感强；❌ 认知度一般 |

### 5.2 最终推荐

#### 🥇 首选推荐：`airoomdesign.com`

**理由**：
1. **SEO 友好**：包含 "AI" + "room" + "design" 三个高价值关键词
2. **扩展性强**：从"户型图"扩展到"室内设计"、"装修设计"自然顺畅
3. **品牌平衡**：既有关键词又有品牌感，不过于直白也不过于抽象
4. **用户理解**：目标用户（设计师、房主、装修爱好者）一看就懂
5. **发音友好**：4个音节，朗朗上口

#### 🥈 备选推荐：`floorplanai.com`

**理由**：
- 如果 SEO 是首要目标，这是**搜索意图匹配度最高**的域名
- "floor plan" + "AI" 直接对应核心搜索词
- 适合**短期快速获取搜索流量**的策略

#### 🥉 品牌向推荐：`planely.com`

**理由**：
- 如果目标是**长期品牌建设**，这是最理想的选择
- 类似 Grammarly、Notionly 的命名模式，易记易传播
- 完全不限制未来扩展方向

## 六、实施建议

### 6.1 域名注册优先级

```
1. 立即查询 airoomdesign.com 可用性 → 注册
2. 同时查询 floorplanai.com 可用性 → 如可用，建议同时注册（防御性）
3. 查询 roomlayoutai.com / smartfloorplan.com → 备选
```

### 6.2 域名迁移 checklist

当前代码中 `aifloorplangen.com` 已硬编码，迁移时需注意：

| 文件 | 位置 | 修改项 |
|------|------|--------|
| `index.html` | `<link rel="canonical">` | 更新为新域名 |
| `index.html` | `og:url` | 更新为新域名 |
| `index.html` | `twitter:image` | 更新为新域名 |
| `index.html` | JSON-LD `url` | 更新为新域名 |
| `sitemap.xml` | 所有 `<loc>` | 更新为新域名 |
| `robots.txt` | `Sitemap` | 更新为新域名 |
| `vercel.json` | 如有 rewrite | 检查并更新 |

### 6.3 品牌一致性建议

- **Logo 设计**：围绕 "AI + Room + Design" 概念，简洁现代
- **品牌色**：建议使用蓝色（信任/科技）+ 绿色（空间/自然）组合
- **Slogan 方向**："AI-Powered Room Design" / "Design Your Space with AI"

## 七、风险提示

| 风险 | 说明 | 应对措施 |
|------|------|---------|
| 域名被抢注 | 热门关键词域名竞争激烈 | 尽快查询并注册首选域名 |
| 商标冲突 | 需确认域名不侵犯现有商标 | 注册前进行商标检索（USPTO/TMDb） |
| SEO 迁移损失 | 更换域名可能导致短期排名下降 | 做好 301 重定向，提交新 sitemap |
| 用户认知成本 | 新域名需要重新建立用户认知 | 配合社交媒体、内容营销同步推广 |

---

> **域名专家总结**：建议优先注册 `airoomdesign.com`，该域名在 SEO、品牌、扩展性三者间取得了最佳平衡。如果该域名已被注册，则退而求其次选择 `floorplanai.com`（SEO优先）或 `smartfloorplan.com`（品牌优先）。
